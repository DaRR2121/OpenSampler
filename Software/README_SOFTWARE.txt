All software here runs on Windows 7 64bit as of 2015 june 24th

- OpenScad is the software to view the 3d models and export data to 2d DXF for CNC (.scad)
- Inkscape can be used to view 2d design files (.svg and .dxf)
- Arduino v0.23 is used to upload and edit the firmware for the Arduino Mega
- python 2.73 is used to run the control software and data processing software
- py-serial for python 2.73 is required for serial control of the OpenSampler


- software that is not included:
- CNC-toolpath creator (CamBam was used)
- Eagle layout editor to edit the RAMPS shield that is put on the Arduino Mega
